DHCP(Dynamic Host Configuration Protocol) : 호스트에 동적으로 ip 주소를 할당하기 위해 사용하는 응용 계층 프로토콜

DHCP 메시지 포맷 : TAG, LENGTH, VALUE(Option), UDP 프로토콜 사용 TCP/IP 지정안해줘도됌
